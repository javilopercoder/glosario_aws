Glosario de Servicios AWS

Cómputo

Amazon EC2

Proporciona capacidad de cómputo escalable en la nube, permitiendo lanzar y gestionar instancias virtuales.

Amazon Lightsail

Ofrece una forma fácil de lanzar y gestionar servidores privados virtuales (VPS), almacenamiento y redes con un costo predecible.

AWS Lambda

Permite ejecutar código sin aprovisionar ni gestionar servidores. Ejecuta el código solo cuando es necesario y escala automáticamente.

Amazon ECS

Servicio de gestión de contenedores Docker que permite ejecutar aplicaciones en un clúster de instancias EC2.

Amazon EKS

Facilita la ejecución de Kubernetes en AWS sin la necesidad de operar directamente clústeres de Kubernetes.

AWS Fargate

Servicio sin servidor para contenedores que funciona con Amazon ECS y EKS, eliminando la necesidad de gestionar la infraestructura subyacente.

AWS Batch

Facilita la ejecución de trabajos por lotes a cualquier escala, sin necesidad de instalar y gestionar software de procesamiento por lotes.

AWS Elastic Beanstalk

Despliega y escala aplicaciones web y servicios desarrollados con Java, .NET, PHP, Node.js, Python, Ruby, Go y Docker.

AWS Outposts

Ofrece servicios, infraestructura y modelos operativos de AWS en instalaciones locales para una experiencia híbrida consistente.

AWS Local Zones

Extiende los servicios de AWS más cerca de grandes centros de población, industria y tecnología de la información para una latencia más baja.

AWS Wavelength

Integra servicios de AWS en la infraestructura de telecomunicaciones 5G para aplicaciones con latencia ultra baja.

AWS Compute Optimizer

Recomienda recursos óptimos para tus cargas de trabajo para reducir costos y mejorar el rendimiento.

Almacenamiento

Amazon S3

Almacenamiento de objetos escalable con alta durabilidad y disponibilidad, ideal para almacenar y recuperar cualquier cantidad de datos.

Amazon EBS

Almacenamiento en bloques persistente para instancias EC2, diseñado para cargas de trabajo intensivas de lectura/escritura.

Amazon EFS

Sistema de archivos NFS escalable y completamente gestionado para uso con servicios de AWS.

Amazon S3 Glacier

Almacenamiento de archivos a largo plazo con costo reducido y recuperación de datos con latencias más largas.

AWS Storage Gateway

Conecta un entorno local a almacenamiento en la nube, proporcionando almacenamiento en la nube sin interrupciones.

AWS Backup

Servicio centralizado para gestionar y automatizar copias de seguridad en los servicios de AWS y en las instalaciones.

AWS Snowball

Dispositivo de transferencia de datos a escala de petabytes para migración y transporte de datos.

AWS Snowball Edge

Dispositivo con capacidad de computación y almacenamiento integrado para recopilación de datos, procesamiento local y migración.

AWS Snowcone

Dispositivo pequeño, portátil y robusto para el procesamiento y la transferencia de datos en el edge.

AWS DataSync

Servicio para transferir datos entre almacenamiento local y almacenamiento en la nube de AWS.

Bases de Datos

Amazon RDS

Servicio de base de datos relacional gestionado que soporta MySQL, PostgreSQL, Oracle, SQL Server y MariaDB.

Amazon DynamoDB

Base de datos NoSQL gestionada que ofrece un rendimiento rápido y escalabilidad automática.

Amazon Redshift

Almacenamiento de datos en la nube rápido y completamente gestionado para análisis de datos a gran escala.

Amazon ElastiCache

Servicio de caché en memoria completamente gestionado compatible con Redis y Memcached.

Amazon Neptune

Base de datos de grafos gestionada optimizada para almacenar y navegar relaciones complejas entre datos.

Amazon DocumentDB

Base de datos de documentos completamente gestionada compatible con MongoDB.

Amazon Aurora

Base de datos relacional compatible con MySQL y PostgreSQL, construida para la nube, que combina el rendimiento y disponibilidad de bases de datos comerciales con la simplicidad de las de código abierto.

Amazon Keyspaces

Servicio de base de datos escalable, altamente disponible y completamente gestionado para Apache Cassandra.

Amazon Timestream

Base de datos de series temporales rápida, escalable y completamente gestionada para IoT y aplicaciones operativas.

Amazon QLDB

Base de datos de contabilidad transparente, inmutable y criptográficamente verificable.

Redes y Entrega de Contenido

Amazon VPC

Proporciona una red virtual privada en la nube donde puedes lanzar recursos de AWS en una red que defines.

Amazon CloudFront

Red de entrega de contenido (CDN) que distribuye contenido con baja latencia y alta transferencia de datos.

AWS Direct Connect

Ofrece una conexión de red dedicada entre tus instalaciones y AWS para mejorar el rendimiento de la red.

AWS Global Accelerator

Optimiza la disponibilidad y el rendimiento de tus aplicaciones utilizando la red global de AWS.

AWS Transit Gateway

Permite conectar miles de VPC y redes locales a través de un solo gateway.

AWS PrivateLink

Ofrece conectividad privada entre VPC, servicios de AWS y aplicaciones en las instalaciones.

AWS Cloud WAN

Servicio de red de área amplia que simplifica la gestión de redes globales.

AWS App Mesh

Permite monitorizar y controlar comunicaciones entre microservicios en aplicaciones.

AWS Site-to-Site VPN

Conecta su red local a una VPC mediante una conexión VPN IPSec.

AWS Client VPN

Servicio VPN basado en cliente que proporciona acceso seguro a sus recursos de AWS y en las instalaciones.

AWS Network Firewall

Servicio de firewall gestionado para proteger sus VPC de amenazas comunes.

Herramientas de Desarrollo

AWS CodeCommit

Servicio de control de versiones gestionado que facilita la colaboración en código con repositorios de Git privados.

AWS CodeBuild

Servicio de integración continua que compila código, ejecuta pruebas y produce paquetes de software listos para desplegar.

AWS CodeDeploy

Servicio de despliegue automatizado que permite automatizar el despliegue de aplicaciones en varios entornos de computación.

AWS CodePipeline

Servicio de entrega continua que automatiza los pasos necesarios para liberar software.

AWS Cloud9

Entorno de desarrollo integrado (IDE) basado en la nube que permite escribir, ejecutar y depurar código en un navegador.

AWS CodeStar

Proporciona una interfaz unificada para gestionar el desarrollo de software en AWS.

Amazon CodeGuru

Herramienta que utiliza machine learning para identificar problemas en el código y recomendar soluciones.

AWS CLI

Interfaz de línea de comandos unificada para gestionar servicios de AWS.

AWS CloudFormation

Proporciona una forma fácil de modelar y aprovisionar recursos de AWS y de terceros.

AWS CloudTrail

Registra, monitorea y retiene la actividad de la cuenta relacionada con acciones en AWS.

Seguridad, Identidad y Cumplimiento

AWS IAM

Gestiona el acceso a los servicios y recursos de AWS de manera segura.

AWS KMS

Facilita la creación y el control de las claves de cifrado utilizadas para cifrar los datos.

AWS Shield

Servicio de protección contra DDoS que protege aplicaciones ejecutadas en AWS.

AWS WAF

Protege aplicaciones web de ataques comunes que afectan la disponibilidad, comprometen la seguridad o consumen recursos excesivos.

Amazon GuardDuty

Servicio de detección de amenazas que monitorea continuamente para proteger sus cuentas y cargas de trabajo de AWS.

Amazon Inspector

Analiza automáticamente las aplicaciones en busca de vulnerabilidades o desviaciones de las mejores prácticas.

AWS Secrets Manager

Ayuda a proteger los secretos necesarios para acceder a sus aplicaciones, servicios y recursos de TI.

AWS Security Hub

Proporciona una vista integral de sus alertas y estado de seguridad en sus cuentas de AWS.

Analítica

Amazon EMR

Servicio gestionado que facilita el procesamiento de grandes cantidades de datos con frameworks como Apache Hadoop y Spark.

Amazon Kinesis

Facilita la ingesta y el procesamiento de datos de streaming en tiempo real.

AWS Glue

Servicio de extracción, transformación y carga (ETL) gestionado que facilita la preparación de datos para análisis.

Amazon QuickSight

Servicio de inteligencia de negocios (BI) que permite crear visualizaciones interactivas y análisis de datos.

Amazon Athena

Permite consultar datos en Amazon S3 utilizando SQL estándar sin necesidad de administrar servidores.

AWS Lake Formation

Facilita la configuración de un lago de datos seguro en días en lugar de meses.

AWS Data Exchange

Facilita la búsqueda, suscripción y uso de datos de terceros en la nube.

Inteligencia Artificial y Machine Learning

Amazon SageMaker

Facilita la construcción, entrenamiento y despliegue de modelos de machine learning.

Amazon Comprehend

Servicio de procesamiento de lenguaje natural que utiliza machine learning para encontrar relaciones y sentimientos en el texto.

Amazon Rekognition

Facilita el análisis de imágenes y videos con capacidades de reconocimiento facial y de objetos.

Amazon Polly

Convierte texto en habla realista.

Amazon Lex

Construye interfaces de conversación en aplicaciones utilizando voz y texto.

Amazon Transcribe

Agrega capacidades de conversión de voz a texto a sus aplicaciones.

Amazon Translate

Servicio de traducción automática neuronal que ofrece traducciones de alta calidad y rapidez.

Amazon Forecast

Servicio de pronóstico de series temporales completamente gestionado basado en machine learning.

AWS DeepRacer

Aprenda sobre reinforcement learning a través de carreras autónomas.

IoT (Internet de las Cosas)

AWS IoT Core

Permite conectar dispositivos IoT a la nube de manera segura y gestionar la comunicación entre estos dispositivos.

AWS IoT Greengrass

Extiende las capacidades de AWS a dispositivos locales para que puedan recopilar y analizar datos más cerca de la fuente de generación.

AWS IoT Analytics

Facilita el análisis de datos de IoT a gran escala.

AWS IoT Device Defender

Ayuda a proteger su flota de dispositivos IoT.

AWS IoT Events

Detecta y responde a eventos de sensores y aplicaciones IoT.

AWS IoT SiteWise

Recopila, estructura y busca datos de equipos industriales a escala.

Servicios de Migración y Transferencia

AWS DMS

Facilita la migración de bases de datos a AWS de manera segura y con tiempo de inactividad mínimo.

AWS Snowball

Dispositivo de transferencia de datos que permite mover grandes cantidades de datos hacia y desde AWS mediante un dispositivo físico.

AWS Application Migration Service

Simplifica y acelera la migración de aplicaciones a la nube.

AWS Migration Hub

Proporciona una ubicación única para rastrear el progreso de las migraciones de aplicaciones.

Aplicaciones Empresariales

Amazon WorkSpaces

Servicio de escritorio como servicio (DaaS) que proporciona escritorios virtuales en la nube.

Amazon WorkDocs

Servicio de almacenamiento y colaboración de documentos seguro.

Amazon Chime

Servicio de comunicación que ofrece conferencias, llamadas y mensajería.

Amazon Connect

Centro de contacto en la nube fácil de usar para ofrecer un mejor servicio al cliente a un costo menor.

Amazon Pinpoint

Servicio de marketing digital para enviar mensajes a sus clientes a través de múltiples canales.

Servicios de Media

AWS Elemental MediaConvert

Permite convertir archivos multimedia en formatos para la transmisión y reproducción en diversos dispositivos.

AWS Elemental MediaLive

Servicio de codificación de video en tiempo real para la transmisión de eventos en directo.

AWS Elemental MediaPackage

Prepara y protege de manera fiable sus videos para su entrega en Internet.

AWS Elemental MediaStore

Servicio de almacenamiento y origen de videos optimizado para medios.

Realidad Virtual y Aumentada

Amazon Sumerian

Herramienta que permite crear y ejecutar aplicaciones de realidad virtual y aumentada en navegadores web.

Desarrollo de Juegos

Amazon GameLift

Facilita la implementación, operación y escalado de servidores dedicados para juegos multijugador.

Amazon Lumberyard

Motor de juegos 3D gratuito y multiplataforma para crear juegos de alta calidad.

Blockchain

Amazon Managed Blockchain

Facilita la creación y gestión de redes blockchain escalables utilizando los frameworks de Hyperledger Fabric y Ethereum.
